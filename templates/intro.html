Acerca del Detector de Plagio

Introducción

Esta aplicación ayuda a detectar plagio en código Java. Compara la estructura de dos archivos Java y utiliza algoritmos de aprendizaje automático para determinar si el código es lo suficientemente similar como para ser considerado plagio.

El plagio es un problema serio en entornos académicos y profesionales. Involucra tomar el trabajo o las ideas de otra persona y presentarlas como propias. Esta aplicación tiene como objetivo ayudar a combatir este problema proporcionando una herramienta para detectar posibles plagios en entregas de código Java.

Cómo Funciona

La aplicación funciona comparando los árboles de sintaxis abstracta (AST) de dos archivos Java. El AST representa la estructura del código, lo que ayuda a identificar similitudes incluso si el código no es idéntico palabra por palabra.

Pasos:

  • Subir Archivos: Los usuarios suben dos archivos Java para ser comparados.
  • Extraer Nodos AST: La aplicación extrae los nodos AST de los archivos Java.
  • Vectorización: Los nodos AST se vectorizan usando TF-IDF.
  • Cálculo de Características: Se calculan características adicionales como la similitud de Jaccard y la distancia de Manhattan.
  • Predicción: Las características se introducen en un modelo XGBoost preentrenado para determinar si los archivos son plagiados.

Herramientas y Tecnologías

Frontend:

  • HTML, CSS, Bootstrap para el diseño
  • JavaScript para la interactividad

Backend:

  • Flask - un framework ligero de aplicaciones web WSGI en Python

Machine Learning:

  • XGBoost - una biblioteca optimizada de gradient boosting distribuido
  • Scikit-learn - para la vectorización y otras utilidades

Entrenamiento del Modelo

El modelo fue entrenado siguiendo estos pasos:

Preprocesamiento de Datos:

  • Extracción de nodos AST de archivos Java
  • Vectorización de nodos AST usando TF-IDF
  • Cálculo de características adicionales (similitud de Jaccard y distancia de Manhattan)
  • Manejo del desequilibrio de clases usando SMOTE

Entrenamiento del Modelo:

  • División de los datos en conjuntos de entrenamiento y prueba
  • Entrenamiento del modelo XGBoost en el conjunto de entrenamiento
  • Evaluación del modelo usando precisión, recall y puntuación F1

Conclusión

Este proyecto muestra el uso de aprendizaje automático y tecnologías web para construir una herramienta práctica para detectar plagio en código Java. La combinación de extracción de AST, cálculo de características y entrenamiento del modelo con XGBoost proporciona una solución robusta a este problema.
